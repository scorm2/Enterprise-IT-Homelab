# Enterprise IT Homelab

> An enterprise-style Windows infrastructure homelab built to develop practical systems administration, Active Directory and enterprise support skills.

---

## Overview

This repository documents the design, implementation and ongoing development of a Windows enterprise environment built using Windows Server 2022, Windows 11 Pro and VMware.

The objective of this project is to simulate the technologies, processes and troubleshooting expected within a real corporate IT environment while developing practical skills in systems administration and enterprise support.

The environment is continually expanded with new infrastructure projects and is fully documented using change records, incident reports and technical documentation.
