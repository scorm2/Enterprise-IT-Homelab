# Environment Overview

## Purpose

This repository documents an enterprise-style Windows infrastructure homelab built to develop practical IT support and junior systems administration skills.

## Environment

| Component | Details |
|---|---|
| Domain | homeab.local |
| Domain Controller | GT-DC01 |
| Client Workstation | COMP01 |
| Server OS | Windows Server 2022 |
| Client OS | Windows 11 Pro |
| Hypervisor | VMware |
| Primary Focus | Active Directory, Group Policy, File Services, Workstation Management |

## Completed Capabilities

- Deployed Active Directory Domain Services
- Created an enterprise OU structure
- Created departmental security groups
- Joined Windows 11 workstation to the domain
- Configured SMB shares
- Applied NTFS permissions using security groups
- Deployed mapped drives through Group Policy
- Configured corporate wallpaper and desktop restrictions
- Configured domain password policy
- Enabled secure Remote Desktop through Group Policy
- Deployed 7-Zip using Group Policy software installation
- Configured Windows Update policy for workstations

## Key Skills Demonstrated

- Active Directory administration
- Group Policy creation and troubleshooting
- NTFS and share permissions
- Security group-based access control
- VMware virtual machine administration
- Windows Event Viewer troubleshooting
- Software deployment via GPO
- Remote Desktop configuration
- Windows workstation management
